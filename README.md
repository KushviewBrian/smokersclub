# Smoker's Club Website

Fort Wayne's premier smoke shop website - a conversion-optimized, locally-focused, compliance-first website designed to position Smoker's Club as Fort Wayne's premier modern smoke shop. The site leverages black-and-mint branding with chrome accents to create a boutique-level experience that justifies premium pricing while driving foot traffic and building digital authority.

## Features

- **Age Verification Gate**: Full-screen modal overlay with session cookie storage for 24 hours
- **Responsive Design**: Mobile-first approach with breakpoints at 768px and 1024px
- **Live Hours Status**: Automatically updates to show "OPEN NOW" or "CLOSED" based on business hours
- **Google Maps Integration**: Embedded map with custom mint green pin
- **Product Categories**: 5 distinct sections with hover effects
- **Compliance Focused**: Includes tobacco permit information and age verification requirements
- **SEO Optimized**: Schema markup, meta tags, and keyword optimization
- **Mobile Optimized**: Sticky CTA bar for easy access to phone and directions
